Shell Scripts
Script 1: In this shell script, we use the concepts of variables, system commands and echo statements to print date and time , username , kernel version , uptime, software chosen, etc.
Script 2:  In this shell script, we use dpkg to find if Python is installed in
the system or not. The version of Python used and the software that maintains it are also displayed. An error message is displayed if Python is not there in the system. 
Script 3:  In this script, the names of important Linux directories and the space used by them is found. We loop through the array of directory names and find out how much space they use.
Script 4:  In this shell script, we read a log file and then find how many times a keyword is present in the file. We loop over the file using a for loop. Here
The chosen keyword is “error”.
Script 5: In this script, we ask the user three questions about open source and then input answers from them. After that we organise the answers in a single
paragraph. This paragraph is written in a text file.
